## 📥 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/smplcorporate/smplapp.gitt
cd smplapp
```

### 2. Install Dependencies

```bash
flutter pub get
```

### 3. Run the App

Connect a device or start an emulator, then run:

```bash
flutter run
```
